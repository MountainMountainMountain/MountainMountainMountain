- 기존 기상청 날씨누리 api를 사용했을 때 산 봉우리의 날씨가 일치하지 않는 오류가 있었습니다. 고도와 위도가 잘못된 오류라 생각하고 DB를 수정했지만 날씨가 일치하지 않는 오류가 계속 돼서 팀원과 함께 문제점을 찾아보니 산악 날씨 데이터는 따로 제공된다는 점을 알게 되었습니다. ??
